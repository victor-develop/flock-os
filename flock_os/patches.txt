# Versioned patches registry for flock_os.
#
# Patches live under flock_os/patches/<semver>/ and run during `bench migrate`.
# Section markers (configparser ini format) split the run order so Frappe syncs
# DocType tables *between* the two phases:
#
#   1. [pre_model_sync]  — run before DocType tables are created/synced.
#   2. sync_all()        — DocType JSON → tabDocType + table DDL.
#   3. [post_model_sync] — run after tables exist (needed by any patch that
#                          touches a Flock DocType or its data/indexes).
#
# Both flock_os patches below read/write Flock DocType tables, so they MUST be
# post_model_sync (a headerless file defaults every patch to pre_model_sync and
# crashes seed_core_fixtures before `tabFlock Group Type` exists).
